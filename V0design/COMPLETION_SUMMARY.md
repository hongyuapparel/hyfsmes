## 💻 技术栈

### 前端技术

```
Next.js 16
├─ App Router
├─ Server Components
└─ React 19.2

UI Framework
├─ Tailwind CSS v4
├─ shadcn/ui 组件
└─ CSS 变量 + 设计令牌

数据可视化
├─ Recharts（图表）
├─ Lucide React（图标）
└─ 自定义组件

样式系统
├─ CSS 变量定义
├─ OKLch 色彩空间
└─ 响应式 Tailwind
```
