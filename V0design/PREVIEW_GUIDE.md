# 完整项目预览指南

## 🎯 项目概览

这是一个**完整的 Next.js 演示项目**，展示了一个专业的服装生产管理系统。采用**专业冷色调**（蓝色 + 青色）+ **活力强调色**（橙色）的现代设计。

---

## 🎨 设计视觉特点

### 色彩方案

**Light Mode（浅色模式）**
```
背景：#F8FAFC（极浅蓝灰）
文本：#1F2937（深灰黑）
卡片：#FFFFFF（纯白）
边框：#E5E7EB（浅灰）
```

**Colors（功能颜色）**
```
主色 (Primary)：专业蓝 #0052CC
- 按钮、链接、导航活动状态

次色 (Secondary)：活力青 #00D9FF
- 强调元素、提示、图标

强调 (Accent)：橙色 #FF6B35
- 关键行动、警告、重要提示

状态色
- 成功：绿色 #10B981
- 警告：橙色 #F59E0B
- 错误：红色 #EF4444
```

**Dark Mode（深色模式）**
```
背景：#0F172A（深蓝黑）
文本：#F1F5F9（浅白）
卡片：#1E293B（深灰蓝）
边框：#334155（中灰）
```

### 排版系统

**标题**
- H1：36-48px、Bold、颜色：主文本
- H2：28-32px、Bold、颜色：主文本
- H3：20-24px、SemiBold、颜色：主文本

**正文**
- Body：16px、Regular、行高 1.6
- Small：14px、Regular、颜色：次文本
- XSmall：12px、Regular、颜色：弱文本

## 📱 页面详细说明

├─ 密码输入

## 🎨 间距和大小规范

### 卡片设计
```
padding：24px（p-6）
border-radius：8px（rounded-lg）
border：1px solid #E5E7EB（border-border/50）
box-shadow：轻微阴影
```

### 按钮
```
height：40px（h-10 在普通大小）
padding：16px（px-4）
border-radius：6px（rounded-md）
font-weight：600（semibold）
```

### 输入框
```
height：40px（h-10）
padding：12px（px-3）
border-radius：6px（rounded-md）
border：1px solid #E5E7EB
focus：border-color 变为 #0052CC
```

### 间距级别
```
2px（space-0.5）、4px（space-1）、8px（space-2）、12px（space-3）、
16px（space-4）、20px（space-5）、24px（space-6）、32px（space-8）
```

## 📐 响应式断点

```
Mobile：< 768px
Tablet：768px - 1024px
Desktop：1024px - 1280px
Large：> 1280px
```

### 响应式应用

**仪表板**
- Mobile：单列布局，侧边栏隐藏
- Tablet：1-2 列统计卡片
- Desktop：4 列统计卡片，2+1 列图表

**登录页**
- 所有设备：居中卡片，最大宽度 448px

## ✨ 交互效果

### Hover 效果
```
- 卡片：border-color 加深，box-shadow 增强
- 按钮：background-color 变深
- 导航链接：background-color 变浅
- 表格行：background-color 变浅
```

### 过渡效果
```
- 所有过渡：200ms ease
- 颜色变化使用 transition-colors
- 尺寸变化使用 transition-all
```

### Focus 状态
```
- 输入框：border-color 变为主色，outline 为环色
- 按钮：outline 为环色
```

## 📝 使用此设计

### 在项目中复制

1. **颜色变量** → 已在 `globals.css` 中定义
2. **组件代码** → 可从 `/components` 和 `/app` 复制
3. **布局模式** → 参考 `dashboard/page.tsx`
4. **数据结构** → 参考仪表板的数据定义

### 自定义建议

- 修改 `globals.css` 中的颜色值
- 编辑 `components/sidebar-nav.tsx` 中的导航项
- 更新 `app/dashboard/page.tsx` 中的统计数据
- 修改 `components/top-nav.tsx` 中的用户信息
