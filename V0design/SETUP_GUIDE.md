## 🎨 关键设计决策

### 为什么是冷色调？
✅ 传达专业和可信  
✅ 适合企业 B2B 系统  
✅ 减少眼睛疲劳（特别是长时间使用）  
✅ 现代感强  

### 为什么加青色强调？
✅ 避免纯深蓝显得沉闷  
✅ 引导用户注意关键信息  
✅ 增加界面的活力感  
✅ 符合现代设计趋势  

### 为什么采用暗色模式？
✅ 适合 24/7 的生产管理场景  
✅ 减少眼睛疲劳  
✅ 现代 UI 设计的主流  
✅ 提升用户体验

## 💡 设计原则（记住这 4 点）

| 原则 | 含义 | 实践方式 |
|------|------|---------|
| **专业第一** | 冷色主导，传达稳定 | 优先使用蓝色、灰色 |
| **活力融合** | 适度强调，避免沉闷 | 用青色突出关键操作 |

| **易用至上** | 清晰层级，易于理解 | 充分的对比度和空间 |

## 📞 常见问题

**Q: 可以修改颜色吗？**  
A: 可以！编辑 `globals.css` 中的 CSS 变量即可全局更改。

**Q: 字体可以改吗？**  
A: 可以！在 `layout.tsx` 中导入不同的 Google Fonts 字体。

**Q: 如何添加新组件？**  
A: 使用 shadcn/ui 提供的组件库，或自己编写遵循设计规范的新组件。

**Q: 支持亮色模式吗？**
